This is a food ordering system called LET'S EAT that I have developed using ASP.NET Core Web API for the backend, React JS for the frontent
and SQL Server for the database. The features of this application is user authentication where different customers can register and create an account,
login into their account and they can also reset there password or email. Once they are registered they can browse the different food items and their categories
that are available on the website to add to their cart. Once they are satified with their selection they can then place an order and have an option 
to either pickup or have the order delivered along with the ability to track the progress of their order. LET'S EAT also has an admin dashboard that only authorized users can access.
where they can see a the amount of food items on the system, total amount of registered users, number of orders that are pending and the total
sales made by LET'S EAT. The admin can also go ahead and add, delete and update food items to the menu, see a list of customers that are registered
and be able to change their authorization on the system. The admin can also see all the orders and update their status in the system.
